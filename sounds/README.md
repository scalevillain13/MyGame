# Sounds Folder

Use this folder for game audio files.

Recommended structure:

- `sounds/hit/` - regular attack sounds
- `sounds/crit/` - critical hit sounds
- `sounds/death/` - slime death sounds
- `sounds/music/` - background music loops

Suggested file names:

- `sounds/hit/hit-1.wav`
- `sounds/crit/crit-1.wav`
- `sounds/death/slime-death-1.wav`
- `sounds/music/forest-loop.ogg`

## Where to find free sounds

- [Pixabay Sound Effects](https://pixabay.com/sound-effects/) - free SFX/music
- [Freesound](https://freesound.org/) - huge sound library (check license per file)
- [OpenGameArt Audio](https://opengameart.org/art-search-advanced?keys=audio) - game-ready assets
- [Kenney Audio](https://kenney.nl/assets?q=audio) - clean packs for games
- [Mixkit Sound Effects](https://mixkit.co/free-sound-effects/) - free effects

## Search keywords

- hit: `slash`, `impact`, `punch`, `whoosh`
- crit: `critical hit`, `stinger`, `spark burst`
- slime death: `slime pop`, `goo splat`, `monster death`
- background music: `fantasy forest loop`, `dark ambient loop`
